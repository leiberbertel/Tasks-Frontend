# Tasks App Frontend
## _This is my frontend app coded with vue 3 that consumes the Tasks Restful API._

## Install dependencies ⚒
```
npm install
```

### Running the project 🚀
```
npm run serve
```

### Lints and fixes files ❌
```
npm run lint
```
